# EVE Frontier Wallet Dashboard

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Built on Sui](https://img.shields.io/badge/Built%20on-Sui-0088FF.svg)](https://sui.io/)
[![Tech Stack: Next.js](https://img.shields.io/badge/Tech%20Stack-Next.js-000000.svg)](https://nextjs.org/)

A unified Web3 dashboard built on the Sui blockchain for EVE Frontier players, enabling seamless management of in-game assets (spaceships, NFTs, resources, tokens) and real-time portfolio analytics.

## 🚀 Core Features

- **Sui Wallet Integration**: Connect with Sui Wallet/Martian Wallet in 1-click
- **In-Game Asset Tracking**: Real-time display of EVE Frontier chain assets (NFTs, tokens, resources)
- **Portfolio Analytics**: Profit/loss statistics, transaction history, asset trend charts
- **EVE Frontier Themed UI**: Optimized for game players with intuitive asset categorization
- **Cross-Device Support**: Responsive design for desktop/mobile

## 🛠 Tech Stack

- **Frontend**: Next.js 14 (App Router), React 18, TypeScript
- **Web3 Integration**: `@mysten/sui.js`, `@mysten/dapp-kit` (Sui official SDK)
- **UI/UX**: Tailwind CSS, shadcn/ui, Recharts (data visualization)
- **Deployment**: Vercel / Cloudflare Pages

## 📦 Quick Start

### Prerequisites

- Node.js ≥ 20.9.0 (required for Next.js)
- Sui CLI (optional, for local testing)

### Installation

```bash
# Clone the repo
git clone https://github.com/你的用户名/eve-frontier-wallet-dashboard.git

# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```
